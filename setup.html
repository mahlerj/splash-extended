Deploying to Heroku using the command line

You can also deploy Splash extended on heroku using command line with the steps bellow:

  1. Heroku login (using your Heroku credentials: Email and Password)

    $ heroku login

  2. Clone the repository on GitHub

    $ git clone https://github.com/logiclinegmbh/splash

  3. Create a Heroku application

    $ cd splash-extended
$ heroku create

  4. Install the Heroku Postgress Add-on

    $ heroku addons: add heroku-postgresql:hobby-dev

  5. Install the Heroku Connect Add-on

    $ heroku addons: add herokuconnect:demo

  6. Add buildpacks

    $ heroku buildpacks:add --index 1 https://github.com/heroku/heroku-buildpack-nodejs
$ heroku buildpacks:add --index 2 https://github.com/heroku/heroku-buildpack-java

  7. Deploy to Heroku

    $ git push heroku master

  8. Run the the application:

    • Open the application in a browser:

      $ heroku open

  9. And then you can see your application running in your browser. Please have look at the architecturefor configuration. For further reading you can also take a look at heroku devcenter.
